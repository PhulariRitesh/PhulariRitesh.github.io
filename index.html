To successfully run the DVFS simulation, follow these steps:

  1. First, enter the Burst Time and Arrival Time for each process in their respective input fields.
  2. Click the Add Process button to add the process to the Ready Queue. You can add multiple processes, but they must be added one at a time.
  3. If you need to delete a process, click the Delete Process button. A popup will appear asking for the Process ID. Enter the ID of the process you want to delete and confirm.
  4. After adding all processes, click the Start Simulation button to run the simulation.
  5. During the simulation, you will see an animation demonstrating how the processes are scheduled.
  6. Once the simulation completes, scroll down to view the Result Table, which contains details such as Process ID, Burst Time, Arrival Time, Frequency, Voltage, and Energy Consumption.

Important: Ensure all processes are correctly entered before starting the simulation.
